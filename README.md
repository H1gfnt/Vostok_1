<h3>Программы</h3>
<ul>
<li>Autopilot_ksp.py - Программа, которая реализует автопилот</li>
<li>Hight_diff.py - Программа, которая рассчитывает зависимость между временем и высотой через имеющуюся математическую модель."</li>
<li>Hight_ksp.py - Программа, которая получает данные полета из KSP, записывает их в файл 'hight.txt', строит график зависимости между временем и высотой через полученные данные.</li>
<li>Hight_comparison.py - Программа, которая строит сравнительный график двух предыдущих.</li>
</ul>
